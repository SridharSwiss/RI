AI at Work: Agents & Copilot

Microsoft 365 Copilot Tuning

  • Allows companies to train AI models using their own internal knowledge, improving domain-specific task performance.
  • Agents built with Microsoft 365 Copilot's Agent Builder can take advantage of these tuned models.
  • Access to tuned models is controlled by the same permissions as the underlying training data.
  • Copilot Tuning will be available in June for enterprises with 5,000+ Microsoft 365 Copilot licenses.
  • Use cases include legal firms automating document generation and consulting companies building regulatory compliance Q&A agents.

Microsoft 365 Copilot: Advanced Development Tools for Teams

  • Agent2Agent (A2A) Protocol: Enables secure peer-to-peer communication between agents for complex enterprise tasks.
  • Updated Teams AI Library: New capabilities in JavaScript and C# help build smarter collaborative agents.
  • Agentic Memory: Allows agents to recall previous user interactions for improved contextual awareness.
  • Automated Agent Validation: Ensures agents comply with Store policies and maintain high-quality performance.
  • Agents in Teams Meetings: Developers can create private or group AI agents that interact within meeting environments.

Microsoft 365 Copilot Studio: Pro-code Features

  • Microsoft 365 Agents Toolkit for Visual Studio: A streamlined development suite integrating AI tools like Microsoft 365 Agents SDK and Azure AI Foundry.
  • Microsoft 365 Agents SDK: Enables scalable, multi-channel agents with deep customization and easy deployment across over 10 messaging platforms.
  • Microsoft 365 Copilot APIs: Enterprise-ready APIs supporting fast, secure, and context-aware generative AI experiences (Retrieval API is in preview).
  • Agent Store: A new marketplace for agents from Microsoft, partners, and customer organizations, discoverable across Microsoft 365 Copilot endpoints.
  • Enhanced Power Platform Connector SDK: Toolset for building enhanced Power Platform connectors for agents to interpret structured enterprise data (Preview).
  • Bring Your Own Models (BYOM): Developers can integrate models from Azure AI Foundry (including over 1,900 available models) into their agents (Preview).
  • Microsoft Dynamics 365 data in Microsoft 365 Copilot will enable users to find Dynamics 365 CRM insights (Private preview for a scoped set of customers).

Microsoft Copilot Studio: New Features for Natural Language Agent Development

  • New capabilities for low-code makers to build more advanced AI agents, enhancing collaboration, governance, and automation.
  • Multi-Agent Systems & Open Standards:
    • Multi-Agent Collaboration: Agents built using Copilot Studio, Azure AI Foundry Agent Service, or Microsoft 365 Agents SDK can work together (Private preview).
    • Agent2Agent (A2A) Protocol: Enables agents to discover and collaborate with peer agents while maintaining governance and security (Preview).
  • Governed Data Access & Integration:
    • Model Context Protocol (MCP) Support: Ensures consistent, governed access to external data and models (Generally available).
    • MCP Servers for Dataverse & Dynamics 365: Allows agents to interact with Microsoft Dataverse and Dynamics 365 tools (Private preview).
  • Enterprise Automation & AI Vision:
    • Computer Using Agent (CUA) Technology: Empowers agents to automate tasks across desktop and web applications using AI vision (Preview).
    • Enhanced Toolset: Includes new capabilities for prompts, document processing, deep reasoning, agent flows, connectors, and REST APIs (Preview).
  • Advanced AI Capabilities:
    • Code Interpreter: Enables agents to write and execute Python code for complex tasks (Preview).
    • Operational Database for Agents: Powered by Dataverse, optimizing speed and real-time interactions (Preview).
    • Dataverse Search: Connects structured and unstructured data across Microsoft 365, Dynamics 365, Power Platform, and external systems (Generally available).

Copilot 365 Wave 2 Spring Release

  • An updated Microsoft 365 Copilot app designed for human-agent collaboration.
  • A new Create experience that brings the power of OpenAI GPT-40 image generation.
  • Copilot Notebooks that turn content and data into instant insights and action (Generally available).
  • Copilot Search and Copilot Memory will begin rolling out in June.
  • Researcher and Analyst, first-of-their-kind reasoning agents for work, will roll out to customers worldwide this month via the Frontier program.

Copilot Pages: New Features

  • Mobile Accessibility: Users can now create and edit Pages directly from their mobile devices.
  • One-Click Word Document Conversion: Pages can be instantly converted into Word documents.
  • Improved Search & Filtering: The Pages module in the Microsoft 365 Copilot app now includes enhanced search and filtering options.
  • Interactive Charts & Code Blocks: Users can generate dynamic charts and code blocks within Copilot Chat.
